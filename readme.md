# Quantium Retail Analytics & Store Trial Analysis

## Project Overview

This project was completed as part of the Quantium Retail Analytics Virtual Internship.

The objective of the analysis was to explore customer purchasing behaviour, identify high-value customer segments, and evaluate the performance of trial store layouts using a data-driven control store methodology.

The project combines exploratory data analysis, customer segmentation, statistical testing, and business-focused reporting.

---

## Business Objectives

- Analyze customer purchasing behaviour
- Identify key customer segments
- Evaluate trial store performance
- Compare trial stores against control stores
- Deliver strategic recommendations for future retail rollouts

---

## Project Workflow

### Task 1 — Customer & Transaction Analysis

- Data cleaning and preprocessing
- Outlier detection and removal
- Product and brand feature engineering
- Customer segmentation analysis
- Affluence and lifestage analysis
- Data visualisation and business insights

### Task 2 — Trial Store Analysis

- Control store selection methodology
- Correlation and magnitude distance analysis
- Trial vs control store comparison
- Statistical significance testing
- Visual assessment of trial performance
- Business recommendations

---

## Key Insights

- Trial Stores 77 and 88 demonstrated significant sales uplift during the trial period.
- Store 86 showed increased customer traffic but weaker sales conversion.
- Mainstream and older family segments generated the highest category spend.
- Premium customer groups demonstrated higher average basket value.
- Trial and control stores followed similar trends pre-trial, validating the methodology.
